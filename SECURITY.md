# Politikan'ny Fiarovana (Security Policy)

## Momba ity projet ity
Ity projet ity dia site statique (HTML, CSS, JavaScript) alefa amin’ny GitHub Pages.
Ity projet ity dia fanampiana anareo amin'ny asan-tserasera sy tambazotra hafahafahanareo manaparitaka kokoa ny zavatra ataonareo amin'ny sehatry ny asa izany
Tsy misy serveur, tsy misy base de données.
Fanahiniako tsy nasina login no miditra aminy toy ireny mail, sns...

## Fiarovana ny mpampiasa
- Tsy misy ny fangatahana anarana, email, na mot de passe izahay aho
- Ny QR Code dia mamorona lien fotsiny araka izay ampidirinareo mampiasa azy

## Raha mahita olana ara-piarovana ianao
Raha mahita bug na olana mety hampidi-doza:
- Aza mamorona issue public na mikitika ny kaaodin'ny utilisateur
- Mifandraisa mivantana amin’ny tompon’ny projet amin’ny GitHub

## Faritra voakasika
- Front-end ihany
- HTML / CSS / JavaScript
- Tsy misy backend
- Tsy misy authentification

## Fanamarihana
Misaotra izay rehetra manampy amin’ny fiarovana sy fanatsarana ity projet ity.
